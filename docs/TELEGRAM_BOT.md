# Telegram Admin Bot

Бот — административный адаптер HYDRA. Доступ разрешён только Telegram user ID,
указанному как Admin Chat ID через TUI.

## Интерфейс

Команда `/start` или `/menu` открывает кнопочный Control Center:

- состояние VPS и сервисов;
- AntiDPI: telemetry counters, запуск/остановка и быстрый разбан;
- Fail2ban: состояние jail и число блокировок;
- настройки уведомлений.

Остановка AntiDPI и разбан требуют подтверждения. Команды `/system`,
`/antidpi`, `/fail2ban` и `/unban <ip>` сохранены.

## Уведомления

Независимо переключаются master switch, AntiDPI, Fail2ban BAN, события UNBAN и
системные уведомления. Настройки атомарно сохраняются в state. Выключенная
категория не выполняет HTTP-запрос к Telegram.

## Безопасность

- IP валидируются и передаются внешним программам только как argv;
- динамический текст экранируется для Telegram HTML;
- токен не выводится при HTTP-ошибках;
- тяжёлые status/systemd операции выполняются вне asyncio event loop;
- опасные кнопочные действия требуют подтверждения;
- монитор Fail2ban переживает logrotate.

Бот работает от root. Токен является root-equivalent секретом: не добавляйте
бота в группы и перевыпустите токен при подозрении на компрометацию.
