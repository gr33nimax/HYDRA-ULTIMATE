## 📖 Возможности

### 🔌 Поддерживаемые протоколы
* **VLESS + REALITY (TCP/443)** — надежный обход цензуры с маскировкой под популярные TLS-сайты.
* **VLESS + xHTTP + TLS** — современная маскировка трафика через WebSocket/HTTP upgrade.
* **NaiveProxy** — Chromium TLS fingerprint, защита от активного зондирования и высокая стойкость.
* **Hysteria2** — обфусцированный транспорт поверх UDP (QUIC) для сверхскоростного соединения на плохих каналах.
* **Mieru** — mTLS прокси с добавлением случайного заполнения (random padding), работающий без привязки к домену.
* **AmneziaWG** — обфусцированная версия WireGuard, запускаемая в Docker-контейнере прямо через API клиента Amnezia.

### 📱 Telegram Интеграция (Два отдельных бота)
1. **Admin Panel Bot (`xray-tg-admin`)** — полноценный пульт управления сервером из Telegram:
   * Получение детальной информации о сервере (загрузка CPU, RAM, диска, аптайм, статус процессов).
   * Просмотр статистики трафика по всем протоколам (детализация Xray по пользователям, AWG по пирам, трафик на портах NaiveProxy, Hysteria2, Mieru).
   * Управление пользователями (`/adduser`, `/deluser`).
   * Управление ссылками подписок (`/sub`, `/delsub`).
   * Просмотр журналов логов (`xray`, `fail2ban`, `honeypot`, `naive`, `hysteria2`, `mieru`, `installer`).
   * Статус джейлов Fail2ban и логов ловушки Honeypot.
   * Безопасная авторизация администратора по сгенерированному паролю.
2. **User Config Bot (`xray-tg-bot`)** — бот для автоматической раздачи подписок пользователям:
   * Получение персональной ссылки на подписку (`/config`).
   * Отображение QR-кода подписки (`/qr`).
   * Просмотр личной статистики трафика и лимитов (`/traffic`).

### 📋 Динамическая система подписок
Локальный HTTP-сервер подписок автоматически формирует конфигурации для импорта в популярные клиенты:
* **Base64** — совместимо с v2rayNG, Hiddify, Shadowrocket, Streisand.
* **Clash Meta (Mihomo) YAML** — для Clash Verge, Nyanpasu, Mihomo Party.
* **Sing-box JSON** — для NekoBox и sing-box. Автоматически экспортирует NaiveProxy (тип `naive`) и Mieru (тип `socks`) для корректной работы.

### 🛡️ Интегрированная безопасность
* **AutoBan** — автоматическая блокировка по TLS-фингерпринтам и некорректным TLS-запросам.
* **Fail2ban** — пресечение попыток подбора паролей SSH, nginx и xray.
* **Honeypot-порт** — ловушка, банящая сканеры сетевых портов на 24 часа.
* **GeoIP Блокировка** — ограничение доступа по базам стран (allowlist/blocklist).
* **WARP-обход на уровне ОС (ip route)** — обход блокировок (OpenAI, Netflix) для ВСЕХ протоколов благодаря динамическому DNS-резолвингу и добавлению маршрутов Linux `ip route`.

```bash

```

|---|--------|----------|

| **4** | 📊 Диагностика и Мониторинг | Проверка задержки (TTFB), аудит соединений, просмотр журналов логов, здоровье нод |
| **5** | 🛡️ Безопасность и Автоматизация | AutoBan, GeoIP-фильтр, ротация UUID, **Telegram Admin Panel**, **Telegram User Bot**, Fail2ban, Honeypot |
| **6** | 📡 Telemt MTProxy | Установка защищенного Telegram MTProto-прокси на Rust/Tokio |

| **11** | 🔐 NaiveProxy | HTTP/2 прокси с Chromium TLS-фингерпринтом и защитой от активного пробинга |
| **12** | 🔒 Mieru | mTLS прокси с обфускацией размеров пакетов (random padding) без необходимости домена |
| **13** | 📹 olcRTC | TCP over WebRTC туннель с имитацией видеоконференций |
